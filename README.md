A minimal To-Do List web app built with HTML, CSS, and JavaScript. It lets you add, edit, and remove tasks, with data saved in local storage for persistence.

Tech Stack:
HTML – Page structure
CSS – Styling & layout
JavaScript – Functionality & local storage
